# Treessence
Some trees for Timber lib

## Set up

**build.gradle**

```
repositories {
    jcenter()
}

dependencies {
    implementation 'fr.bipi.treessence:treessence:0.0.1'
}
```

### Usage

  1. Checkout [Timber](https://github.com/JakeWharton/timber)
  2. Add any trees you like
